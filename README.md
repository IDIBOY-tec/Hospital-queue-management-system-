# Hospital-queue-management-system-
project code 60
Hospital Queue Management System – A web-based application developed as a final academic project to improve patient registration, queue management, and service delivery in healthcare facilities.
ospital Queue Management System

Objectives
- Reduce patient waiting time.
- Improve service delivery in hospitals.
- Organize patient queues effectively.
- Enhance record management.
- Improve communication between patients and hospital staff.
 Features
- Patient registration
- Queue number generation
- Queue management
- Staff dashboard
- Appointment management
- Report generation
- User authentication
 Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
Installation
1. Download or clone the repository.
2. Place the project in your web server directory (XAMPP or WAMP).
3. Import the database into MySQL.
4. Update the database connection settings.
5. Start Apache and MySQL.
6. Open the project in your browser.
